NOTES
--------------------------------------------------------------------------------

HOWTO BUILD
	Step 1) Install PhoneGap and any other prerequisite development tools
			for your target platforms of choice.
			See: http://www.phonegap.com/
	Step 2) Navigate to the "phonegap_client" directory in a command shell
			Run the following commands:
			> phonegap build <platform> [e.g. "phonegap build ios"]
			> phonegap install <platform> [e.g. "phonegap install ios"]
	Step 3) Done! You should now be running on-device (if attached & able
			to deploy) or running in a simulator.
